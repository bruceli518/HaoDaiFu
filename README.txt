[General description] This dataset is crawled from the websit 'www.haodf.com', which is the largest online diagnosis platform in China.
[Detail description] The dataset contains 65535 rows and 3 columns. Each row stands for one diagnosis record of one patient. The 
'label' column is the label for each record. The 'description' column is the detail description of the patient conditions, it is equal to main
complaint in formal hospital health records.The 'heading' is the summary of 'description'.
